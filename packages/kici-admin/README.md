# kici-admin

Administration CLI for a KiCI customer-deployable orchestrator. Manage runs, agents, peers, sources, secrets, the event log, and the database schema — and install the orchestrator itself as a system service.

Part of [KiCI](https://kici.dev) — CI/CD workflows as TypeScript code: author them with full language power, dry-run them locally, and run them on your own infrastructure.

## Install

```bash
npm install -g kici-admin
```
